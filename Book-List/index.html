Book List

小说

德米安，[德] 赫尔曼·黑塞著，姜乙译，天津人民出版社

老实说，我不知道怎么评价这本书，因为读完这本书我确有收获，但心中却感觉这收获只是黑塞想借这本书传达给我的一小部分而已。可遗憾的是，以我现在的年龄经历，好像也只能得到这些了。从直观感受来讲，这本书的主题重点在两个字——自我上，黑塞好像在教我认识自我，告诉我什么是爱，如何正视自己的感受，无惧于孤独、困苦与迷茫，尽情探索自己追寻的东西并为之奋斗终身。
整本书读下来很舒服，语言优美，充满了诗意和哲理，没有一定的经历，其中蕴含的哲理未必都能读懂。在读的过程中，脑海中总是会感慨：黑塞到底是在写小说还是在写诗啊，哈哈。
总之，虽未得甚解，但还是很喜欢这本书，期待过一段时间之后再读时，自己能获得的感受。

悉达多，[德] 赫尔曼·黑塞著，姜乙译，天津人民出版社

本书篇幅不长，读起来很快，语言优美，哲理性强，读起来总会让人禁不住思考。个人在读此书时，不仅能从书中读出一些佛教禅宗哲学，也能体会到一些道教思想，甚至还会让我想起存在主义和唯物主义等等。
但无论如何，我想黑塞想告诉读者的应该是：世界是真实的，爱也是真实的，而你，我的朋友，你要做的事情是倾听自己内心的声音，去找寻自己想要的东西。
值得一提的是，书中某些话语简直写出了个人的心声，唯一遗憾的就是没有早点读到此书。

罪与罚，[俄] 陀思妥耶夫斯基著，朱海观、王汶译，人民文学出版社

此书发表于俄国农奴制改革后，各种思想、主义充斥在当时俄国人民的心中，本书的主人公正是如此。而本书讲述的也就是这个满腔热血、受过高等教育（法学）且拥有着自己的一套“犯罪”理论的青年在冲动下杀人的故事。作者一共围绕着两个主题展开——罪与罚（这也是本书的名字）。其中，罪是事情的经过，而罚则是事情的动机、心理以及后续的自首、愧疚、爱和救赎等等。就个人而言，最认可的是本书中作者对人物心理的描写，而作者借着人物之口发表的对社会、犯罪的看法在当时也应该会使人耳目一新，倒也值得让人思考一番。另外，本书还有一个很关键的细节，也就是宗教，可以说在当时的俄国，宗教在一定程度上影响着人们的行为。最后，这本书强烈推荐年轻人读一下，因为刚步入社会的年轻人，突然间面对着社会中的种种事情，一定会产生很多与在学校时不一样的想法，本来就是血气方刚的，一旦想不开就容易干傻事。

局外人，[法] 阿尔贝·加缪著，柳鸣九译，上海译文出版社

这本书不长，读起来也很快，不过语言精炼，内容饱满，而且作者想表达的东西也很直观。从整个故事来讲，默尔索被判死刑是可以意料的（毕竟他杀了人），但不合理的地方在于他被判死刑的原因——仅仅是因为人们认为他是一个冷血的人（或者说他表现冷血），而人们对他认知的来源也仅仅是因为他在母亲的葬礼上表现的不够伤心（这是因为在他看来，人的死亡是在正常不过的事情了）。可是，人们却忽略了默尔索曾经与母亲一起生活的日子，事实上，他并没有做过任何对不起母亲的事情。
人总会生老病死，死者安息，生者继续前行，这是自古以来的“规则”。默尔索平静的接受了母亲的死亡，一个人仍然过着自己的生活，丝毫不做任何多余的事情，即便是面对法官的审问，他也丝毫不掩饰自己的想法，不会用虚伪的谎言为自己辩护，率性而为，带着一种天真，即便是死亡也无法让他屈服，这就是加缪想表达的“存在主义”——人是在无意义的宇宙中生活，人的存在本身也没有意义，但要尊重人的个性和自由，而人也可以在原有存在的基础上自我塑造、自我成就，活得精彩，从而拥有意义。

黑铁时代，王小波著，北京十月文艺出版社

这本书收录了王小波去世前的手稿，包括有些已经发表的小说、未完成的小说和一些随笔。从内容上来讲，这本书应该算做读完小波的其他作品后的“补读本”，因为其中有些文章是他对自己作品的一些说明，读起来有点类似电影的幕后花絮的感觉，建议这本最后再看~🙂
不过可惜的是这本书里未完成的小说，还没来得及写就走了...

白银时代，王小波著，北京十月文艺出版社

这一版的白银时代只有三篇：《2010》、《2015》和《白银时代》，少了一篇《未来世界》。这本书一如王小波的风格——戏剧性、黑色幽默和讽刺，个人感觉《2010》最有意思，《2015》和《白银时代》有些的地方读的不太懂，有些地方又读出了自己，结果就是笑不出来，反而有点欲哭无泪了，不过换个角度，自己也觉得自己很好笑😂。尽管书中尽是些荒诞、戏剧的描写，但是现实生活往往就是如此。

黄金时代，王小波著，北京十月文艺出版社

这一版的黄金时代只有三篇：《黄金时代》、《三十而立》和《似水流年》，且这三篇的质量与《白银时代》的三篇差不多，但这三篇所写的内容既不同、也更大胆，因为这三篇时不时的会来点少儿不宜的内容（笑ing）。每当自己读到这些内容时，首先感受到的是好笑，因为那个时代的人们对待性可能真的很“僵硬”（现在人们对待性更加理智和客观），读起来就会很诙谐；其次感受到的是王小波借助性所表达的东西，有时是纯粹的情感表达，有时是对待固化思维的讽刺，有时又是对形式主义的挖苦等等。实际上，他之所以能这么写，是因为他认为性也是生活的一部分，也是人这种生物的一部分，本来就需要表达，为啥还要遮遮掩掩呢？另外，这本书字里行间读起来还是一如既往的给人一种很坦诚的感觉，再加上时不时的黑色幽默，还是会让人偶尔会忍俊不禁😂，但读者能否从中获得乐趣，这又得取决于读者自身了。

呼兰河传，萧红著，湖南文艺出版社，博集典藏馆版

前不久刚把《呼兰河传》看完，其中“荒凉”一词的出现频率很高，基本覆盖了所有的章节。其中有一章中连续地好几小节都会出现这样一句话：“我家是荒凉的”，甚至有一小节就只有这样一句话，当时觉着这个女人的怨念极深。后来发现，她也许是在发泄，好似在对已经接受了的充斥着苦难和悲伤的现实发牢骚。发泄过后，她又依然的追求着她那“永恒的憧憬和追求”。

从她的文字中可以读出，她是天真且阳光的，更重要的是她所追求的事物是勇敢的。

电影《黄金时代》两年前就已经欣赏过了，正好的得知了写给她的歌曲《只得一生》，便听了一番。《只得一生》内有一歌词：“不懂你的为你忧愁，明白你的叹此生值得一游”，之前觉得这句歌词写的很有意思，让人感觉好像其中有着很多故事或者感情，值得让人去了解。现在看来，像是对她的安慰。

杂文

我的精神家园，王小波著，北京十月文艺出版社

这本书是由一篇篇随笔跟杂文构成的，涉及的内容就是王小波本人对生活方式、电影、媒体、文艺工作等的看法以及国内外见闻的思考。他喜欢用简单直接的比喻（比喻可能会有些糙），用幽默的语气讲出一些道理，这可能就是他向读者构建的精神家园。读这本书的时候，就如同在和他对话，感受着他的思考。

沉默的大多数，王小波著，北京十月文艺出版社

这本书记录的是王小波的一篇篇杂文和随笔，涉及的内容就是王小波本人对思想、人性、写作等的看法以及国内外见闻的思考。他喜欢用简单直接的比喻（有些比喻会比较恶心😂），用幽默的语气讲出一些道理，时常让人忍俊不禁，心里不由得嘀咕：这个王小波真皮🤣。读这本书的时候，既不乏趣味性，也不少思维性，对于同样工科出身的我而言，读起来就如同在听一个多年未见的老朋友讲着这些年的所见所闻、所思所感。

一只特立独行的猪，王小波著，北京十月文艺出版社

同样是王小波的随笔、杂文、跋语和序等，语言风格一如既往，读起来的感觉依然是直率坦然，即便是一只猪，也要当一只特立独行的猪😛。

散文

我与地坛，史铁生著，人民文学出版社

本书是史铁生的散文集，收录了《我与地坛》、《合欢树》等以记事为主的散文，主要内容包括了个人经历、人生感悟和一些文革时期的事情。其描写语言生动，感情细腻，一字一句中充满了对生活、生存、人生、亲情的哲思，特别是将身处逆境之中的心理状态描写的十分真实。在阅读过程中，脑袋中经常会冒出一个想法：要是早点读到这本书就好了。当一个人处于迷茫时期的时候，我真心推荐读一下这本书。🙂

访谈

我们唱，叶三著，台海出版社

这本书不能用太严格的角度来看待，行文的风格比较随意散漫，有点像随笔，又有点像访谈。就内容来讲，本书主要记录了作者对国内一些音乐人的访谈，可以向读者或听众呈现一个真实的音乐人，所以，当成故事书看会更好。
另外，这本书是朋友送给我的。

传记

断头艳后，[奥] 斯蒂芬·茨威格著，陈祚敏译，北京出版社

这本书是在我大学时代读的，也是我读的第一本茨威格的书。借助茨威格、陈祚敏的视角，可以看到一个真实存在的玛丽·安东瓦内特。茨威格总是喜欢从人的角度来思考问题，所以，在他的笔下，历史事件仿佛变得更易于让人理解了。通过窥探王后的人生，我们也能从中获得一些启示，对她所经受的苦难和折磨，也表示深刻的同情。但是历史的齿轮需要她的推动，需要很多个这样的她（他）推动。

这本书个人很推荐一读，二手书市场能淘到这本书就可以收藏一本了。

话说，当时做的读书摘录还夹在我的另一本书中。

计算机

算法笔记，胡凡、曾磊著，机械工业出版社

这本书很适合零基础、非计算机专业而想从事计算机行业的人读，内容非常通俗易懂，不过也需要自己下功夫理解。也许是因为本书的作者就是非科班出生的缘故，在介绍一些知识的时候总是会尽可能的将这些东西讲的通俗。此书配套的还有对应的《算法笔记上机训练实战指南》。

C++ Primer 中文版（第 5 版），[美] Stanley B. Lippman / [美] Josée Lajoie / [美] Barbara E. Moo，电子工业出版社

这本书的名字真的是一言难尽，要问哪门编程语言最复杂，首选就是 C++ 。因为这本 C++ “初级读本”都是特么七八百页的😑，一开始接触编程的人怎么可能一下子懂那么多东西。所以，这本书不如叫做 C++ Dictionary。这么叫其实也凸显出了这本书的一大优点——这是本工具书，内容十分详细，查漏补缺很有帮助。另外，这本书还有一个特点，那就是这本书更多的是在介绍编译器的行为，并没有涉及到更深层次的东西（好吧，也许这就是叫做 Primer 的缘故吧😂）。总而言之，这是 C++ 程序员绕不开的一本书，常备在手边，Debug 的时候仿佛有了三位“大护法”，哈哈。

C程序设计语言（第2版·新版），[美] Brian W. Kernighan / [美] Dennis M. Ritchie，机械工业出版社

这本 C 语言之父写的两百多页的“小册子”（相比其他计算机丛书而言）相当不错，语言简洁明了（正如 C 语言的特性：简单而高效），例子形象生动，更重要的是，工程设计思想在本书中也有体现，真的十分不错😤。个人并不推荐完完全全的初学者读这本书，它适合有了一定 C 语言基础的初学者、进阶者或熟练者读。当然了，个人也建议多读几遍这本书，从个人读完后的感受来看，初学者、进阶者、熟练者读这本书的收获应该都是不相同的，再次建议多读几遍。最后要说明下，这本是介绍以 ANSI C 标准（与 C99 类似，但不尽相同）介绍 C 语言，所以某些地方的设计可能已与现在不符，但重点在于其中的工程设计思想而不是代码实现本身。

心理

异类，（美）马尔科姆·格拉德威尔著，中信出版社

这本比尔盖茨推荐过的书，确实有点意思。作者站在与前人不一样的角度看待人的成功，并指出了其中的因果关系，是很独特的看待世界的方式。另外，这本书不但对个人具有一定的启示作用，而且对子女教育的观念也有很大的影响（也许更甚之）。不同的人读过之后，应该都会有点收获。所以，推荐阅读。

音乐

民谣吉他考级标准教程，王鹰、马鸿著，漓江出版社

这本书被很多人吐槽内容不好、错误多等。但是个人感觉，这本书确实是对的起这个价格了（手动滑稽）。尽管 A 调入门确实有点赚噱头的意思，但是其讲述的知识点是没有问题的，每一级也是循序渐进的。而且，这本书的语言其实很简洁明了。不管是介绍一些基本的技巧，还是吉他的乐理，讲的都很通俗易懂。所以，用来当作琴行教材是没问题的，但是需要一个靠谱的老师😂。但不太建议当作个人入门自学的教材，倒是可以作为个人练习的书目。另外，这本书在编配的时候也是尽量遵照原版、好弹的原则编配的，所以这也是一个优点。不过，有些谱子的定级有点问题。当然，这对不考级的人没有影响。

社会学

乡土中国，费孝通著，北京时代华文书局

这本书是费老先生上个世纪四十年代在西南联大的讲学之作，总共 14 篇，篇篇短小精悍，语言简单朴实，却不失精确深沉。尽管书中的有些观点已经不符合现代社会了，但这本书仍然对了解中国社会文化的基本特性具有重要的参考意义和价值。第一次读这本书，就对书中宗、族概念的理解叫好，难以想象在抗战胜利之前就有学者对中国社会的研究到了如此深入的地步了。
目前读完一遍，只能理解一部分，相信等自己更成熟一点之后，重读会有更多的体会。另外，只要生活在中国社会，这本书也应该要好好读一读，对做人做事应该都有些许帮助。
